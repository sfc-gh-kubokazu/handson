# Step 4: Personal Skills — カスタムスキルの作成

**所要時間:** 20分  
**目的:** Workspace にカスタムスキルを作成し、定型業務を `/` コマンドで呼び出せることを体験する。

> **注意:** Personal Skills は Snowsight の Workspace 機能です。  
> スキルは `.snowflake/cortex/skills/` ディレクトリに保存され、`/` コマンドで呼び出せます。

---

## Personal Skills とは

> スキルは Cortex Code への「専門知識の追加」です。  
> 定型的な作業手順をスキルとして定義しておくことで、毎回同じプロンプトを書かずに済みます。

```
/monthly-report   ← 月次売上レポートを自動生成
/data-quality     ← データ品質チェックを実行
```

---

## 1. スキルの作成方法（2通り）

### 方法A: UI から作成（推奨）

1. Workspace のサイドパネルで **Cortex Code パネル** を開く
2. パネル下部の **スキルアイコン（/）** をクリック
3. **+ Create Skill** を選択
4. スキル名と内容を入力

### 方法B: ファイルを直接作成

1. Workspace 内に `.snowflake/cortex/skills/` ディレクトリを作成
2. スキルファイル（`.md`）を配置

---

## 2. 月次売上レポートスキルを作成する

以下の内容でスキルを作成:

**スキル名:** `monthly-report`

**スキル内容（SKILL.md）:**

```markdown
# 月次売上レポート生成

## 概要
月次の売上サマリーレポートを生成します。

## 手順
以下のステップで月次売上レポートを作成してください:

1. 指定月（デフォルト: 先月）の売上データを集計
   - 実店舗売上合計
   - EC売上合計  
   - 前月比
   
2. 商品カテゴリ別売上ランキング（トップ5）

3. 売上上位商品（トップ10）

4. 結果を読みやすい Markdown 形式でまとめる

## データソース
- SNOW_RETAIL_DB.SNOW_RETAIL_SCHEMA.RETAIL_DATA
- SNOW_RETAIL_DB.SNOW_RETAIL_SCHEMA.EC_DATA
- SNOW_RETAIL_DB.SNOW_RETAIL_SCHEMA.PRODUCT_MASTER
```

---

## 3. スキルを呼び出す

Cortex Code のメッセージボックスで `/` を入力すると、作成したスキルが表示されます。

```
/monthly-report
```

または

```
/monthly-report 2026年4月分のレポートを作成して
```

---

## 4. 別のスキルを作ってみる（時間があれば）

**スキル名:** `data-quality`

```markdown
# データ品質チェック

## 概要
指定テーブルのデータ品質を確認します。

## チェック項目
1. NULL値の割合（列ごと）
2. 重複レコードの確認
3. 外れ値の検出（数値列）
4. 参照整合性チェック

結果をテーブル形式でまとめてください。
```

呼び出し:

```
/data-quality RETAIL_DATA のデータ品質を確認して
```

---

## ポイント

- スキルはチームで共有できる「定型業務の自動化テンプレート」
- 一度作ればメンバー全員が同じクオリティのアウトプットを出せる
- スキルファイルは Git で管理できるため、バージョン管理も可能
