## トラブルシューティング早見表

| 症状 | 対処 |
|---|---|
| ログインできない | URL 確認 / パスワードリセット |
| ウェアハウスが起動しない | ロールが `SYSADMIN` であることを確認 |
| COPY INTO で 0 files | テーブルを TRUNCATE してから再実行 |
| `junior_dba` でエラー | accountadmin で権限付与してから切り替え |
| タイムトラベルで復元できない | 保持期間（Standard: 1日）を超えていないか確認 |
